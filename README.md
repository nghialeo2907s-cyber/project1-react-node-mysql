Project: React + NodeJS + MySQL + Docker

Mục tiêu:
Thiết lập môi trường lập trình sử dụng Docker cho:
- Frontend: React (build & serve bằng Apache)
- Backend: NodeJS + Express
- Database: MySQL 8

Yêu cầu
- Docker Desktop (WSL2)

Cách chạy project

1. Clone source
```bash
git clone https://github.com/nghialeo2907s-cyber/project1-react-node-mysql
cd project1-react-node-mysql



2. Khởi chạy Docker

docker compose up --build


3. Truy cập hệ thống

- Frontend (React): http://localhost:8080

- Backend API (Node): http://localhost:3000
